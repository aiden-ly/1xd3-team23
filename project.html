Client Project - StudyFlow

Client Name: Jonathan DiCecco, Computer Science Society

Target Users:

  • University students, particularly first-years who are adjusting to academic life
  • Users are likely to use both desktop and mobile versions of the app
  • May have a preference for mobile due to moving around to go to classes.

Static Information:

  • App name: "StudyFlow"
  • About page explaining the app's purpose
  • FAQ section for common questions
  • Basic tutorial/guide for new users
  • Contact form for support

Design Elements:

  • Clean, minimalist interface
  • Calming color scheme (blues and whites)
  • Intuitive navigation
  • Avoid bright colors and cluttered layouts
  • Simple, non-distracting animations

Data Management:

  • User profiles (name, major, semester)
  • Course list and tasks/assignments
  • Due dates and priority levels
  • Course-organized notes
  • Calendar events
  • Task progress tracking
  • Task completion statistics
  • Study time logs

Simple Interactive Elements:

  • Pomodoro timer
  • Progress bars for tasks
  • Drag-and-drop task organizer
  • Calendar widget
  • Productivity trend charts
