Client Name: Jonathan DiCecco

App Name: StudyFlow


Purpose

It is a productivity app that is designed to help university students, but can be used by anyone, to help manage their classwork and time. We plan to include features such as a to-do list, a built-in Pomodoro timer, an assignments list sorted by priority, dark mode, trend charts to track your progress, and a calendar view for all tasks.


Data Description

Login Info/User Profile: A list of all login information to allow users to have their own individual tasks. Their username and password will be used to login, while the rest of the information inputted will be displayed at their profile. Fields: Username (TEXT, PRIMARY), Password (TEXT), Real Name (TEXT), School (TEXT), Major (TEXT), Semester (TEXT)


To-do List Table: A simple to-do list to be used daily. It will store each task that the user adds (this can be anything and doesn't have to be related to a specific course/assignment) with a checkbox, showing if it is completed or not. Fields: Name (TEXT, PRIMARY), Description (TEXT), Completed (BOOLEAN).


Tasks/Assignments: A list of all of the tasks the user adds so that they can easily see an overview of what they need to do and which assignments to tackle first. It will store each task which will each have their own due date and priority. Fields: Name (TEXT, PRIMARY), Description (TEXT), Course (TEXT), Due Date (DATE), Priority (TEXT), Completed (BOOLEAN).
